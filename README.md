# Medical Insurance Charges Prediction

**AI Intermediary Level Project**

Simple linear regression project that predicts medical insurance costs using age, BMI, number of children, smoking status and region.

# Most important finding

**Smoking has by far the strongest effect**  
→ smokers pay approximately $23651 per year

# What was done

- Basic data exploration & visualization
- Converted categories to numbers (one-hot encoding)
- Trained Linear Regression model
- Measured error: RMSE= **$5,796** on test data

- Python
- pandas, numpy
- matplotlib, seaborn
- scikit-learn
